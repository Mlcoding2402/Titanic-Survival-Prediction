# Titanic Survival Prediction

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Algorithms Tested
- Logistic Regression: 81.0%
- Decision Tree: 77.1%
- Random Forest: 83.8%
- KNN: 65%
- SVM: 59%

## Best Model
Random Forest (83.8% Accuracy)
